####
This is a movie database website built using React. Users can explore a wide range of movies, watch trailers directly on YouTube, leave comments about the films, and view movie posters. The site offers an intuitive and engaging user experience, making it easy for movie enthusiasts to discover and discuss their favorite films.

####
The data for this site is fetched using Axios to communicate with a backend application developed with Spring Boot, which is hosted at <a href="https://github.com/renanmello/movieback-end">here</a>.
